# Fonts

React Native requires TTF or OTF font files (not woff2).

On your Mac, download the TTF versions:
- bebas-neue.ttf      → https://fonts.google.com/specimen/Bebas+Neue
- dm-mono-400.ttf     → https://fonts.google.com/specimen/DM+Mono (Regular 400)
- dm-mono-500.ttf     → https://fonts.google.com/specimen/DM+Mono (Medium 500)  
- dm-sans-400.ttf     → https://fonts.google.com/specimen/DM+Sans (Regular 400)
- dm-sans-500.ttf     → https://fonts.google.com/specimen/DM+Sans (Medium 500)

Place all TTF files in this directory, then the app will load them correctly.
